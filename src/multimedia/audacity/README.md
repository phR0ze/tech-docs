## Change speed of audio
1. Launch `audacity`
2. Select `File >Import >Audio...`
3. Select all audio with `Ctrl+a`
4. Select `Effect >Pitch and Tempo >Change Tempo...`
5. Change the `Percent Change: -15.00` and click `Apply`
