# Harness <img style="margin: 6px 13px 0px 0px" align="left" src="../../data/images/logo_36x36.png" />

A harness is the orchestration and tooling surrounding an LLM and is just as important as the LLM
itself in some regards. There are many variations from chat bots, to CLIs with system wide access to
dekstop GUI apps.
