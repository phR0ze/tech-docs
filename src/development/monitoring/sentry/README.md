# Sentry <img style="margin: 6px 13px 0px 0px" align="left" src="../../data/images/logo_36x36.png" />

Sentry is an enterprise grade monitoring tool for:
* Error Monitoring
  * Alerts and notifications via email or other intergrations
* Session Replay
* Tracing
* Code Coverage
